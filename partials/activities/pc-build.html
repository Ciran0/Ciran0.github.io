Building my Own Computer

Any game rite of passage

Building my own computer has been on my bucket list for a very long time, but it's only as an IT student that I finally got the motivation (and money) to tackle this project

Who would have believed, making an analysis

The first step of the process was to really try and gain a deeper understanding of how a computer works, how the different parts interact together, what would fulfill my needs as a gamer and what would fit into my budget. Multiple ressources like ltt tutorials, the r/pcmasterrace community on reddit and online component selector were a big help. I also had the chance of beeing able to ask advice to colleagues from Ephec to make sure that my build was sensible

Replacing the powersuply and increasing storage

Building the computer was a really rewarding process but something that really validated my decision to learn this skill was when i needed to repaire or upgrade my setup. Around a year after the build the power supply broke and with my newfound skills and cofidence (and amazon fast shipping) I was able to diagnose the issue, select a new, fitting power supply and install it in a record time. And when, as a proud game myself, came the day to upgrade my setup I had no issues doubling the Ram of my system or installing a new M2 SSD.

Skills that translate

When my father found an old 50euro computer I had no issues identifying the components, diagnosing issues and ultimately installing a new wifi card into the motherboard of the computer. This allowed me to pretty easily set up a small debian server under my desk

Skills Gained

  • Deeper understanding of modern computer components and their interactions
  • Hardware troubleshooting skills and the hability to fix/upgrade a system
  • Confidence and knowledge required to navigate specsheets of computer components
  • Ability to navigate OS installation, BIOS updates and boot process debuging

This learning process was extremely rewarding, not only by the fact that it ended with me having a new computer, but also because this allowed be to gain a deeper understanding of how things work. And ultimately the hability to control the tools I use every day in a more complete way
